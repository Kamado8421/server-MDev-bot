GET /get/user/

Faz a busca de um usuário no banco de dados a partir do número de telefone dele.

fetch('/get/user/?jid=123456789', {
    method: 'GET'
})
.then(response => response.json())
.then(data => console.log(data));

GET /get/config/

Obtém as configurações do bot determinadas pelo dono [prefixo, nome-bot, etc.].

fetch('/get/config/?identificadorNomeBot=myBot', {
    method: 'GET'
})
.then(response => response.json())
.then(data => console.log(data));

GET /get/apikey/

Obtém uma chave API de algum sistema externo a partir do nome dado à chave e sua senha.

fetch('/get/apikey/?apiname=myApi&password=secret', {
    method: 'GET'

.then(response => response.json())
.then(data => console.log(data));

POST /post/create-user/

Rota para criar um novo usuário no banco de dados.

fetch('/post/create-user/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        jid: '123456789',
        nome: 'Novo Usuário',
        xpInicial: 0,
        saldoInicial: 100
    })
})
.then(response => response.json())
.then(data => console.log(data));

POST /post/editUserAttribute/

Rota para editar um atributo de um usuário a partir de seu jid.

fetch('/post/editUserAttribute/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        jid: '123456789',
        editarAtributo: 'saldo',
        novoValor: 200,
        adicionarNoAtributo: false
    })
})
.then(response => response.json())
.then(data => console.log(data));
