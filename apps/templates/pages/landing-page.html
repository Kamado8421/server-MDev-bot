Qual a linguagem de programação do MDev-Bot?

Para aconexão do script com o WhtasApp atulizamos NodeJs. Ele é responsável por gerenciar a intermace do WhtasApp e envios de mensagens através de uma API. Para o controle de banco de dados usamos Python.

O script do bot é gratuito?

Sim e não. O código do bot está disponível gratuitamente, mas por está em desenvolvimento ele permanece criptografado. Mas não fique triste!! O código do bot é ensinado no nosso canal do YouTube

Pode fazer um comando X?

Claro!! vá no último video ou shorts no canal do YouTube e dê a sugestão.

O bot vai continuar sendo atualizado?

Isso depende muito de vocês na aréa do engajamento do projeto e sugestões de comandos.

Como faço para mudar o prefixo do bot?

Se você tá lendo isso, já é meio caminho andado. Faça Login nessa página com seu usuário de administrador para encontrar essa opção.

Como criar um usuário admin na plataforma?

Há um video no canal do YouTube ensinando. Tenha o Python instalado. localize no terminal a pasta em que está o servidor do bot, use os comados respectivamente: "python manage.py makemigrations", "python manage.py migrate" "python manage.py createsuperuser". Após o último comando insira os dados pedidos para o novo admin: usuario, email e senha.
